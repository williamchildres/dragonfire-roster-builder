# Contributing

Thanks for helping improve Dragonfire Roster Lab.

## Development

```bash
npm install
npm run lint
npm run test
npm run build
```

Keep changes focused, add tests for behavior changes, and do not commit generated `dist` files.

## Data Contributions

All combat data must include source evidence. Unknown values should remain unknown until verified. See `docs/CONTRIBUTING_DATA.md`.

## Assets And Branding

Do not add official logos, extracted game assets, copied dragon art, or copyrighted artwork. Use original UI elements or properly licensed open-source icons.
